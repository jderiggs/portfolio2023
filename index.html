PolyTools

UI/UX & 3D Design

Polytools is a low-level design prototype, and UI/UX solution for a mobile application storefront. It is involved with the display, customization and sale of procedural parameter driven textures for 3D applications. Since they are fully procedural, this allows users to customize values such as colour and details within the textures by imputing numerical values. This can be used for rendering, game design or any other 3D application that renders textures.

SetupXR

UI/UX & User Research

The main objective of SetupXR is to address the issue of limited leg mobility and lack of accessibility within existing Virtual Reality applications. The goal is to create a mid-level UI/UX solution that is easy to use and enables disabled individuals to set up alternate means of achieving room-scale functionality (i.e., crouching, walking, and standing) within VR.

Material Studies

3D Design

A suite of procedurally generated parameter-driven materials created using Adobe Substance 3D Designer and rendered in Marmoset Toolbag 4. They can be used in any 3D design workflow, from rendering to game design, and are created by me for practice and the texturing of models for Product Design or Game Design. These textures take into account the physical properties of the material such as roughness, metallic and height.
